# Detector
Object detection on static images.

Repository consist of 2 files:
index.html
sketch.js

and is using:
Bootstrap for CSS
ML5.js for COCO-SSD model
P5.js for images processing

Usage:
Open picture file with high resolution, and press DETECT button. That's it. ;)
